# 🍞 Тёплого хлеба — клубное пространство пекарни

Веб-приложение для сообщества пекарни «Тёплого хлеба» — хлебный тиндер, обратная связь и соседский чат.

## Основные разделы

### 1. С кем разделить тёплого хлеба?
Хлебный тиндер — поиск единомышленников по интересам и любимым позициям:
- Два режима: **«Разделить фильтр»** (друзья) и **«Купить хлеб домой»** (тиндер)
- Матчинг по ключевым словам и любимой выпечке
- Свайпы (лайк/дизлайк) с уведомлением при взаимной симпатии
- Фильтрация по полу

### 2. Влияйте на «Тёплого хлеба»
Форма обратной связи:
- **«Хочу пожаловаться»** — форма жалобы
- **«Расскажу о хорошем»** — форма похвалы
- История своих отзывов

### 3. По соседству
Визуализация закрытого Telegram-чата сообщества с реалистичными сообщениями.

## Аккаунт (клубная карта)

- Регистрация по номеру клубной карты и псевдониму
- Профиль: фото, пол, возраст, дата рождения, описание
- Календарь мероприятий с записью
- Бонусная система
- История покупок

## Стек

- **Backend:** Python, Django 5, SQLite
- **Frontend:** HTML, CSS (кастомный, Tinder-inspired дизайн)
- **Шрифты:** Playfair Display, Inter

## Запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Миграции
python manage.py migrate

# Наполнение демо-данными (15 пользователей + 4 мероприятия)
python seed_data.py

# Запуск сервера
python manage.py runserver
```

Открыть: **http://127.0.0.1:8000/**
